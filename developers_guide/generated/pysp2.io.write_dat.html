pysp2.io.write_dat#

pysp2.io.write_dat(ds, file_name)[source]#

This writes the .dat files that generate the intermediate parameters used by the Igor processing.

Parameters:

  • ds (xarray Dataset) – The dataset containing the processed signal data.

  • file_name (str) – The name of the file to save to.
